Initial release of @lessgo/plugin-vite - Vite plugin for less.go

Features:
- Compile LESS files using the fast less.go (lessc-go) compiler
- Full TypeScript support with comprehensive type definitions
- Source map support (enabled by default in development)
- HMR (Hot Module Replacement) support for development
- Configurable include/exclude patterns for file filtering
- Support for LESS plugins, global variables, and modify variables
- Automatic node_modules resolution for @import statements
- Proper error formatting for Vite's error overlay
